AgenticKanbasutra

Plan, queue, run, and review coding-agent tasks across local Git repositories with visible status, runner configuration, artifacts, and failure recovery.

View on GitHub Report an Issue

Workflow

Keep agent work visible from idea to review.

  1. 01Create

    Define repository, branch, runner, model, permissions, priority, and context.

  2. 02Queue

    Run tasks manually or let automatic mode respect your concurrency limit.

  3. 03Execute

    Dispatch manual handoffs, generic CLI commands, Copilot, Claude, or Codex runners.

  4. 04Review

    Inspect summaries, logs, prompts, changed files, diffs, and failed-task details.

Runners

Use the right execution path for each task.

manual

Safe handoff for interactive tools and preview validation.

generic-cli

Custom local automation through a configurable command template.

copilot-cli

Local GitHub Copilot CLI attempts with captured output.

claude-cli

Local Claude Code CLI execution with configurable permission arguments.

codex-cli

Local Codex CLI execution with sandbox-aware permission mapping.

cloud handoffs

Preview dispatch paths for hosted provider workflows.

Safety

Designed for review, not blind automation.

AgenticKanbasutra can launch local commands and provider CLIs. Keep secrets out of task specs, prefer worktree isolation for write-capable runs, and review generated changes before accepting them.

  • Explicit permission profiles
  • Manual and automatic queue modes
  • Prompt, stdout, stderr, summary, and diff artifacts
  • Failed-task visibility and quick requeue workflows
